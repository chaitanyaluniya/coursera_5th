# Paradise Nursery - Plant Shopping Application

## Project Description
Paradise Nursery is a React-based web application designed for plant enthusiasts. This project features a landing page, a product listing page with various categories of plants, and a functional shopping cart managed using Redux Toolkit.

## Key Features
* **Landing Page:** An inviting entry point with a "Get Started" button.
* **Product Listing:** A variety of plants categorized by use (e.g., Air Purifying, Fragrant).
* **Shopping Cart:** Users can add plants to their cart, update quantities, and remove items.
* **State Management:** Powered by Redux for a seamless shopping experience.

## Tech Stack
* React
* Redux Toolkit
* Vite
* CSS3
